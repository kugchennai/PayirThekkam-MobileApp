## About

PayirThekkam is a mobile application designed to help farmers easily find nearby storage facilities to safely store their produce after harvest.
The goal is to reduce post-harvest losses by giving farmers quick access to verified storage locations, availability details, pricing, and directions.

This project was built during the Kohacks Buildathon using modern Kotlin Multiplatform technologies.

## Key Features

* Locate nearby storage facilities based on user location
* View storage capacity, availability, and pricing
* Map-based navigation to each facility
* Unified Android and iOS app using Compose Multiplatform
* Shared Kotlin business logic and UI components
* Clean, modern interface and extendable architecture
